# Ghost Summoner App

Ghost Summoner is a Next.js web application. This repository contains the source code and instructions for local development and deployment.

## Development

Requirements
- Node.js (v16 or later recommended)
- npm or yarn

Install dependencies
```bash
npm install
# or
yarn
```

Run development server
```bash
npm run dev
# or
yarn dev
```

Build and run production
```bash
npm run build
npm run start
# or
yarn build
yarn start
```

Firebase Hosting (deployed)
This project can be deployed to Firebase Hosting. You can deploy a Next.js app using Firebase Hosting with Cloud Functions or the new Firebase Hosting build integration. See Firebase docs for details.
